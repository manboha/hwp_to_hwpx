# HWP → HWPX 일괄 변환기

한컴오피스 COM 자동화를 이용해 지정 폴더의 `.hwp` 파일을 `.hwpx` 형식으로 일괄 변환하는 Python GUI 앱입니다.

---

## 주요 기능

- 폴더 지정 → 하위 폴더 포함 `.hwp` 파일 자동 탐색
- 변환 전 대상 파일 목록 미리보기
- 파일별 진행 로그 및 프로그레스 바
- 이미 `.hwpx`가 존재하는 파일 자동 건너뜀
- 별도 저장 폴더 지정 가능 (미지정 시 원본 위치에 저장)
- 파일 접근 허용 보안 팝업 자동 차단 (`FilePathCheckerModule` 등록)

---

## 요구사항

| 항목 | 내용 |
|---|---|
| OS | Windows |
| 한컴오피스 | 2022 이상 (한글 설치 필수) |
| Python | 3.10 이상 |
| 패키지 | `pywin32` |

> 한컴오피스가 설치되어 있어야 COM 자동화가 작동합니다.  
> macOS / Linux에서는 실행되지 않습니다.

---

## 설치

```bash
pip install pywin32
```

---

## 실행

```bash
python hwp_to_hwpx.py
```

---

## 사용 방법

1. **HWP 파일이 있는 폴더** 선택
2. (선택) **저장 폴더** 지정 — 비워두면 원본 파일 옆에 저장
3. **목록 새로고침** 버튼으로 변환 대상 확인
4. **변환 시작** 클릭

---

## 참고: HWPX 형식이란?

HWPX는 한컴오피스의 오픈 문서 형식으로, 내부 구조가 ZIP + XML입니다.  
구형 바이너리 HWP와 달리 텍스트 추출·파싱이 용이하여  
문서 아카이빙, 지식 베이스 구축, AI 활용 등에 적합합니다.

## 알려진 제약

- 한글 97 이하 버전으로 작성된 파일은 변환되지 않을 수 있습니다.
- 암호화(잠금) 문서는 변환에 실패할 수 있습니다. 암호화 문서는 [한컴 공식 HWPX 변환기](https://www.hancom.com)를 이용하세요.
